# Project Context

Este projeto foi criado para resolver o desafio técnico da First Answer.

Objetivo:

Receber um texto e uma marca monitorada.

Retornar:

{
  "marca_monitorada_encontrada": true,
  "outras_marcas": []
}

Stack:

- Python 3.11
- Groq API
- Streamlit
- Docker
- Docker Compose
- Pydantic
- Pytest
- Ruff
- Black
- Isort
- Pre-commit

Arquitetura:

- Clean Architecture
- SOLID
- Dependency Injection

Documentação:

Ler todos os arquivos da pasta docs/prds antes de implementar qualquer código.
